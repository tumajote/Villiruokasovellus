# Villiruokasovellus
Sovellus mahdollistaa luonnosta löytyvään ravintoon (esim. villiyrtit, marjat, sienet, riista ja kalansaaliit) liittyvän tiedon tallentamisen ja käsittelyn. Käyttäjä voi tallentaa löytämänsä villiruoan lajin, määrän sekä löytöajankohdan ja -paikan. Käyttäjä voi valita näkyykö merkintä vain itselle vai onko se julkinen. Käyttäjä voi myös jakaa löytönsä jollekin toiselle käyttäjälle, eli valita kenelle muista käyttäjistä hänen löytönsä näkee. Jokaisella käyttäjällä on henkilökohtainen näkymä, josta voi tarkastella omia merkintöjään sekä lajeista ja paikoista tehtäviä yhteenvetoja.
